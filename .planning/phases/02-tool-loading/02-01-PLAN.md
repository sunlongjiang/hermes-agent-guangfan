    文件头部：
    ```python
    """Extract and write tool descriptions from hermes-agent tool files.

    Parses Python source files in hermes-agent/tools/ to extract tool schema
    descriptions (top-level and per-parameter). Supports format-preserving
    write-back so evolved descriptions can be saved without breaking schema structure.
    """

    import ast
    import re
    from dataclasses import dataclass, field
    from enum import Enum
    from pathlib import Path
    from typing import Optional

    from rich.console import Console

    console = Console()
    ```

    **DescFormat 枚举:**
    ```python
    class DescFormat(Enum):
        SINGLE_LINE = "single_line"       # "description": "text"
        PAREN_CONCAT = "paren_concat"     # "description": ("a " "b ")
        TRIPLE_QUOTE = "triple_quote"     # "description": \"""text\"""
        VARIABLE_REF = "variable_ref"     # "description": VAR_NAME
    ```

    **ToolParam dataclass:** (per D-07)
    字段: name(str), type(str), required(bool=False), enum(Optional[list[str]]=None), description(str=""), desc_format(DescFormat=SINGLE_LINE), desc_line_offset(int=0)
    方法: to_dict() 返回 {"name", "type", "required", "enum", "description"}, from_dict(cls, d) 恢复实例

    **ToolDescription dataclass:** (per D-07)
    字段: name(str), file_path(Path), description(str), params(list[ToolParam]=field(default_factory=list)), desc_format(DescFormat=SINGLE_LINE), schema_var_name(str=""), raw_source(str="")
    方法: to_dict() 返回 {"name", "file_path"(str), "description", "params":[p.to_dict()]}, from_dict(cls, d) 恢复实例

    **discover_tool_files(hermes_agent_path: Path) -> list[Path]:**
    扫描 hermes_agent_path / "tools" 下的 *.py 文件，返回内容中包含 "registry.register(" 的文件列表（自动排除无工具定义的辅助模块）。用 sorted() 保证顺序稳定。

    **核心提取函数 extract_tool_descriptions(file_path: Path) -> list[ToolDescription]:**

    解析策略（分层，per D-01）:
    1. 读取文件全部内容 `source = file_path.read_text()`
    2. 用正则 `r'^([A-Z][A-Z0-9_]*_SCHEMA(?:S)?)\s*=\s*[\[{]'` (MULTILINE) 找到所有 schema 常量名和位置
    3. 对每个 schema 常量:
       - 如果以 `= [` 开头（list 模式），使用 brace/bracket 计数提取整个 list 文本范围，然后在其中按 `{` 分割出每个 schema dict
       - 如果以 `= {` 开头（dict 模式），用 brace 计数提取单个 schema dict 文本范围
    4. 对每个 schema dict 文本块:
       - 用正则 `r'"name"\s*:\s*"([^"]+)"'` 提取 name
       - 用 `_extract_description_at()` 提取 top-level description（在 `"description"` key 后、`"parameters"` key 前的区域搜索）
       - 进入 `"properties"` 块，对每个 property 用相同逻辑提取 param description
       - 用正则提取 `"type"`, `"required"`, `"enum"` 字段构建 ToolParam

    **_extract_description_at(source: str, pos: int) -> tuple[str, DescFormat, int, int]:**
    从 `"description":` 后面的位置开始:
    - 跳过空白
    - `(` 开头 -> DescFormat.PAREN_CONCAT: 找匹配的 `)`, 用 `ast.literal_eval()` 求值
    - `"""` 开头 -> DescFormat.TRIPLE_QUOTE: 找匹配的 `"""`, 用 `ast.literal_eval()` 求值
    - `"` 或 `'` 开头 -> DescFormat.SINGLE_LINE: 找闭合引号（处理转义）, 用 `ast.literal_eval()` 求值
    - 大写字母开头 -> DescFormat.VARIABLE_REF: 提取变量名, 在文件中搜索 `VAR_NAME = ` 定义并递归求值

    **_find_matching_bracket(source: str, start: int, open_char: str, close_char: str) -> int:**
    从 start 位置开始计数括号层级，处理字符串内的括号（跳过引号内内容），返回匹配的闭合位置。

    **_extract_properties(source: str, properties_block: str, file_source: str) -> list[ToolParam]:**
    在 `"properties"` dict 内按 `"param_name":` 分割，对每个 param 提取 type, description, enum, 再结合 `"required": [...]` 列表判断 required 状态。

    错误处理: 文件读取失败或无 schema 定义时返回空列表，解析单个 schema 失败时跳过该 schema 并用 console.print 输出警告。

    ```python
    import os
    from evolution.core.config import get_hermes_agent_path

    HERMES_AVAILABLE = False
    try:
        _hermes_path = get_hermes_agent_path()
        HERMES_AVAILABLE = _hermes_path.exists()
    except FileNotFoundError:
        pass

    @pytest.mark.skipif(not HERMES_AVAILABLE, reason="hermes-agent not available")
    class TestRealHermesAgent:
        def test_discover_finds_tool_files(self):
            """discover_tool_files returns at least 15 files (22 known)."""
            path = get_hermes_agent_path()
            files = discover_tool_files(path)
            assert len(files) >= 15
            # 不应包含辅助文件
            names = [f.name for f in files]
            assert "__init__.py" not in names
            assert "registry.py" not in names

        def test_extract_memory_tool(self):
            """memory_tool.py has paren_concat description with 4 params."""
            path = get_hermes_agent_path()
            tool_file = path / "tools" / "memory_tool.py"
            tools = extract_tool_descriptions(tool_file)
            assert len(tools) >= 1
            memory = next(t for t in tools if t.name == "memory")
            assert "persistent memory" in memory.description.lower()
            assert memory.desc_format == DescFormat.PAREN_CONCAT
            assert len(memory.params) >= 2
            # 验证 frozen fields
            action_param = next(p for p in memory.params if p.name == "action")
            assert action_param.type == "string"
            assert action_param.enum == ["add", "replace", "remove"]
            assert action_param.required is True

        def test_extract_all_tools_no_crash(self):
            """All discovered files extract without exceptions."""
            path = get_hermes_agent_path()
            files = discover_tool_files(path)
            total_tools = 0
            for f in files:
                tools = extract_tool_descriptions(f)
                total_tools += len(tools)
            # 至少 30 个工具（已知约 50 个含 browser 的 10 个）
            assert total_tools >= 30
    ```
