Implement `evolution/prompts/prompt_dataset.py` following the ToolDatasetBuilder pattern exactly.

**PromptBehavioralExample** (per D1):
- `@dataclass` with fields: section_id (str), user_message (str), expected_behavior (str), difficulty (str, default "medium"), source (str, default "synthetic")
- `to_dict()` returns dict of all fields
- `from_dict(cls, d)` filters to known dataclass fields, returns instance

**PromptBehavioralDataset** (mirrors ToolSelectionDataset):
- `@dataclass` with train/val/holdout lists of PromptBehavioralExample
- `all_examples` property: concatenates all three lists
- `save(path: Path)`: creates dir, writes train.jsonl/val.jsonl/holdout.jsonl (one JSON per line)
- `load(cls, path: Path)`: reads JSONL files, returns dataset
- `to_dspy_examples(split, section_texts=None)`: converts to `dspy.Example` with `task_input=user_message`, `expected_behavior=expected_behavior`. If section_texts dict provided and section_id matches, adds `section_text` field. Marks `task_input` as input via `.with_inputs("task_input")`.

**PromptDatasetBuilder** (per D1, D2):
- Nested `class GenerateSectionScenarios(dspy.Signature)` with InputFields: section_text, section_id, num_scenarios, difficulty_mix. OutputField: scenarios (JSON array string).
- `SECTION_WEIGHTS` class constant: `{"default_agent_identity": 20, "memory_guidance": 15, "skills_guidance": 15, "platform_hints": 20, "session_search_guidance": 10}`
- `__init__(self, config: EvolutionConfig)`: stores config, creates `dspy.ChainOfThought(self.GenerateSectionScenarios)` predictor
- `generate(self, sections: list[PromptSection]) -> PromptBehavioralDataset`:
  1. Group sections by category (platform_hints.* sections share the "platform_hints" weight, divide 20 across sub-keys: 20 // num_platform_keys per key, remainder to first keys)
  2. For each section, call predictor with section text, section_id, target count, difficulty_mix "easy:30%,medium:50%,hard:20%"
  3. Parse JSON array from output, create PromptBehavioralExample instances
  4. Shuffle all examples, split by config.train_ratio / config.val_ratio / remainder
  5. Return PromptBehavioralDataset

Use two-stage JSON parsing (try json.loads, fall back to brace-extraction) matching dataset_builder.py pattern.
