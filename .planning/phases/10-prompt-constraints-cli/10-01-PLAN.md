    1. Copy `_parse_bool()` function from `tool_constraints.py` verbatim -- same conservative strategy.

    2. Create `PromptRoleChecker` class:
       - Inner class `RoleCheckSignature(dspy.Signature)` with docstring explaining role preservation check:
         - `section_id: str = dspy.InputField(desc="Section identifier (e.g. memory_guidance)")`
         - `original_text: str = dspy.InputField(desc="Original section text before evolution")`
         - `evolved_text: str = dspy.InputField(desc="Evolved section text to check")`
         - `role_preserved: bool = dspy.OutputField(desc="True if evolved text maintains the same functional role as original")`
         - `explanation: str = dspy.OutputField(desc="Explanation of role assessment")`
       - CRITICAL: Boolean direction is POSITIVE -- `role_preserved=True` means PASS (opposite of ToolFactualChecker's `has_false_claims=True` means FAIL).
       - `__init__(self, config: EvolutionConfig)`: store config, create `self.checker = dspy.ChainOfThought(self.RoleCheckSignature)`.
       - `check(self, section_id: str, original_text: str, evolved_text: str) -> ConstraintResult`:
         - Create LM via `dspy.LM(self.config.eval_model)`.
         - Call `self.checker()` inside `dspy.context(lm=lm)`.
         - Parse `_parse_bool(result.role_preserved)`.
         - If True: `ConstraintResult(passed=True, constraint_name="role_preservation", message=f"Role preserved in '{section_id}'", details=explanation)`.
         - If False: `ConstraintResult(passed=False, constraint_name="role_preservation", message=f"Role changed in '{section_id}'", details=explanation)`.
       - `check_all(self, original_sections: list, evolved_sections: list) -> list[ConstraintResult]`:
         - Build `original_map = {s.section_id: s for s in original_sections}`.
         - For each evolved section, find original by section_id. Skip if no match.
         - Call `self.check(evolved.section_id, original.text, evolved.text)`.
         - Return list of results.
