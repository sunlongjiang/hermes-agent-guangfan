    **`evolve()` function** with params: `section=None, iterations=10, eval_source="synthetic", hermes_repo=None, dry_run=False`:

    Steps (numbered, with Rich output, symmetric with evolve_tool_descriptions.py):

    1. **Configuration**: Create `EvolutionConfig(iterations=iterations)`. If hermes_repo, set `config.hermes_agent_path = Path(hermes_repo)`.

    2. **Extract sections**: Construct `prompt_builder_path = config.hermes_agent_path / "hermes_agent" / "prompt_builder.py"`. Call `extract_prompt_sections(prompt_builder_path)`. Exit if empty. Display Rich Table with section_id, char_count columns.

    3. **Dry-run gate**: If dry_run, print summary (section count, dataset source, iterations), return.

    4. **Build PromptModule**: `module = PromptModule(original_sections)`. Keep `original_sections` as copy for diff/constraint comparison.

    5. **Generate/load dataset**: If "synthetic": `PromptDatasetBuilder(config).generate(original_sections)`, save to `datasets/prompts/`. If "load": `PromptBehavioralDataset.load(Path("datasets") / "prompts")`.

    6. **Per-section GEPA optimization** (KEY DIFFERENCE from tool CLI):
       - `sections_to_optimize = [section] if section else module._section_ids`
       - Validate section exists if specified (`if section and section not in module._section_ids: error`).
       - For each `active_sid` in `sections_to_optimize`:
         - `module.set_active_section(active_sid)`
         - Filter dataset: `section_train = [ex for ex in dataset.train if ex.section_id == active_sid]`
         - Same for val.
         - Build `section_texts = {s.section_id: s.text for s in original_sections}`.
         - Create temporary `PromptBehavioralDataset(train=section_train, val=section_val, holdout=[])`.
         - Call `.to_dspy_examples("train", section_texts=section_texts)` and `.to_dspy_examples("val", section_texts=section_texts)`.
         - If trainset empty, skip this section with warning.
         - Configure LM, run GEPA: `dspy.GEPA(metric=metric, max_steps=iterations).compile(module, trainset=trainset, valset=valset)`.
         - Fallback to MIPROv2 on exception (same pattern as tool CLI).
         - Update module reference with optimized result.

    7. **Extract evolved**: `evolved_sections = module.get_evolved_sections()` (or `optimized_module.get_evolved_sections()`).

    8. **Constraint validation** (per D4: after GEPA, before holdout):
       - `validator = ConstraintValidator(config)`
       - For each evolved section: `_check_growth(evolved.text, original.text, "prompt_section")`, `_check_non_empty(evolved.text)`.
       - `role_checker = PromptRoleChecker(config)`, `role_checker.check_all(original_sections, evolved_sections)`.
       - Print results with color icons. If any fail, save to `output/prompts/FAILED_{timestamp}/` and return.

    9. **Holdout evaluation**: For each holdout example, run baseline module and optimized module forward, collect outputs. Use `PromptBehavioralMetric(config)` to score. Compute baseline vs evolved average scores.

    10. **Report**: Rich Table with Baseline Score, Evolved Score, Change, Section Count, Time, Iterations.

    11. **Save results** to `output/prompts/{timestamp}/`:
        - `evolved_sections.json`: list of `{"section_id": ..., "text": ...}`.
        - `metrics.json`: timestamp, iterations, scores, improvement, section count, elapsed, etc.
        - `diff.txt`: output of `_generate_diff()`.

    **Click CLI**:
    ```python
    @click.command()
    @click.option("--section", default=None, help="Section ID to optimize (default: all sections)")
    @click.option("--iterations", default=10, help="Number of GEPA iterations per section")
    @click.option("--eval-source", default="synthetic", type=click.Choice(["synthetic", "load"]),
                  help="Source for evaluation dataset")
    @click.option("--hermes-repo", default=None, help="Path to hermes-agent repo")
    @click.option("--dry-run", is_flag=True, help="Validate setup without running optimization")
    def main(section, iterations, eval_source, hermes_repo, dry_run):
        """Evolve hermes-agent prompt sections using DSPy + GEPA optimization."""
        evolve(section=section, iterations=iterations, eval_source=eval_source,
               hermes_repo=hermes_repo, dry_run=dry_run)

    if __name__ == "__main__":
        main()
    ```
